# Restaurant Bazi API

โปรเจคจบ ทำร้านอาหารแนะนำเมนูผ่าน Lineliff

## Features

- การลงทะเบียนผู้ใช้พร้อมการคำนวณดวงชะตา
- การทำนายดวงรายวันโดยใช้ AI
- การบริหารจัดการร้านอาหาร

---

## Tech Stack
- **React – UI library** - สำหรับสร้าง SPA
- **Vite – Build tool & dev server** 
- **Zustand** - Global state management
- **React Router DOM** - Client-side routing สำหรับ SPA
- **Axios** - HTTP client สำหรับติดต่อ backend API
- **LINE LIFF SDK** - @line/liff
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Icon library
- **ESLint** - Code quality & linting
- **PostCSS / Autoprefixer** - CSS processing
- **Environment Variables**
- **Node.js** + **Express.js** - Backend framework
- **MySQL** - Database
- **bcrypt** - Password hashing
- **express-validator** - Input validation
- **Groq AI** - Fortune predictions
- **Thailand Bazi API** - Birth chart calculations

---

# Install Frontend And Backend 

## Frontend Install
```bash
npm install
npm install react-router-dom zustand axios lucide-react cors
```

## Backend Install
```bash
npm install
npm install nodemon mysql2 bcrypt axios dotenv express multer cloudinary multer-storage-cloudinary pg
```
